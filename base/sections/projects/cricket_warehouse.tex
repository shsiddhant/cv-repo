\resumeProjectHeading{Cricket Warehouse $|$ \href{https://github.com/shsiddhant/cricket-warehouse}{Github}}
  {March 2026}
  {Python, PostgreSQL, dbt}
  \begin{itemize}
    \item Built a scalable ball-by-ball data warehouse for cricket analytics using Python, PostgreSQL, and dbt.
    \item Engineered an incremental ETL pipeline that ingests complex, nested JSON data from Cricsheet, utilizing file hashing to prevent duplicate processing.
    \item Modeled warehouse layers using dbt (staging → intermediate → marts).
    \item Built normalized relational tables for matches, players, teams, innings, and deliveries.
    \item Developed a Python CLI tool for database configuration, data fetching, ingesting, and pipeline management.
    \item Created analytical marts enabling statistical queries and downstream modeling.

  \end{itemize}
